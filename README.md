## üöÄ Quick Start

### Prerequisites
- Node.js 18+ (recommended: 20+)
- pnpm (or npm/yarn)
- A Polkadot wallet extension

### Installation

\`\`\`bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
\`\`\`

Open your browser to the URL shown in terminal (usually http://localhost:5173)
